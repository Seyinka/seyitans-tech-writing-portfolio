## Getting Started with Mautic

This guide walks you through installing, setting up, and running Mautic on your local or server environment.

---

## Prerequisites

Before installation, ensure you have:

| Requirement | Recommended Version |
|--------------|--------------------|
| PHP | 8.1 or higher |
| MySQL/MariaDB | 5.7+ / 10.4+ |
| Composer | Latest |
| Web Server | Apache or Nginx |
| Git | Installed |
