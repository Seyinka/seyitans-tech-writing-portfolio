# Overview: Amazon S3

Amazon Simple Storage Service (**Amazon S3**) is a cloud-based object storage service.  
Think of it as a hard drive in the cloud where you can store files (called *objects*) in folders (called *buckets*).  

## Why Use S3?

- **Simple**: Easy to start — just create a bucket and upload files.  
- **Scalable**: Stores any amount of data.  
- **Secure**: Fine-grained access control.  
- **Flexible**: Can serve as storage, backup, or even host a static website.
